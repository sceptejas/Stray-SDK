
## Testing & CI/CD

[![CI Pipeline](https://github.com/sceptejas/Stray-SDK/actions/workflows/ci.yml/badge.svg)](https://github.com/sceptejas/Stray-SDK/actions/workflows/ci.yml)

Stray SDK includes comprehensive testing infrastructure:

- **23 automated tests** covering all core functionality
- **GitHub Actions CI/CD** for automated testing
- **Multi-version support** (Python 3.8-3.12)

### Running Tests Locally
```bash
pytest tests/ -v
```

See [TESTING.md](TESTING.md) for detailed testing documentation.
